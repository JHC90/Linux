# Archivieren und Komprimieren

* **tar Archivierung**
  * Einpacken / Create File = cf
   >tar -cf tarfilename file1 dir2

  * Anzeigen Inhalt TarFile / tf
   >tar -tf tarfilename

  * Entpacken Tar-File
  >tar -xf tarfilename
 

 
* **gzip / bz2 Komprimierung**
  * Einpacken / Create File = cf
   >gzip File
   >bzip File
  * Entpacken 
  >gunzip *.gz
  >bunzip *.bz2

* **Auspacken tarball**
  > tar -xzf *.tar.gz
  > tar -xjf *.tar.bz2
